The immensely popular Google Font service is dead simple, and displays type beautifuly. Widespread use of the Google Font API on the web also means lots of cross-site caching. Everyone wins when users can avoid re-downloading fonts and speeding up load times.

Google hosts over 600 fonts in their library.

Unfortunately, most of them are downright terrbile. There are however, a few notable exceptions which are presented below.

The most difficult task facing a new user of the directory is simply sorting out the high-quality typefaces from the fluff.

This website features the lyrical mastery of rapper Kendrick Lamar, set to a curated selection of the highest quality Google Fonts.

Clicking on an example will re-direct you to the Google web fonts directory.
